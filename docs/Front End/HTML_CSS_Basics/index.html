Front End - HTML & CSS Basics

Step 1

Understanding the HTML document layout

<!DOCTYPE html>

<!-- This is the root tag of the document -->
<html>
  <!-- This is the head tag of the document. 
    This is primarily where imports (CSS and JS usually) are located-->
  <head>
  </head>

  <!-- This is the body tag of the document.
    This is where all of the visual elements of the document live. -->

  <body>
    Here's some random text content!
  </body>
</html>

Step 2

Adding more HTML text

<!DOCTYPE html>

<html>
  <head>
  </head>
  <body>

    <!-- This is a div tag. Think of it like a folder/directory - it's for organization. -->
    <div>

      <!-- A 'h1' tag is a text element formatted to be the biggest header -->
      <h1>Here's a title!</h1>

      <!-- A 'p' tag is a paragraph of text -->
      <p>Here's some text content!</p>

      <!-- A 'h2' tag is a text element formatted to be the second biggest header -->
      <h2>Here's a smaller title!</h2>

      <p>Here's some more text content!</p>
    </div>
  </body>
</html>

Step 3

Basic CSS and adding complex content

<!DOCTYPE html>

<html>
  <head>
    <style>
      /* Apply style to everything with class "doggies" */
      .doggies {
        max-width: 300px;
      }
    </style>
  </head>
  <body>

    <!-- Add a picture wrapped in a div -->
    <div>
      <img class="doggies" src="../res/doggies.jpg">
    </div>

    <div>
      <h1>Here's a title!</h1>
      <p>Here's some text content!</p>
      <h2>Here's a smaller title!</h2>
      <p>Here's some more text content!</p>
    </div>
  </body>
</html>

Step 4

More complex CSS styles

<!DOCTYPE html>

<html>
  <head>
    <style>
      /* Apply style to everything in the body - in this case, a font */
      body {
        font-family: Helvetica, Arial, sans-serif;
      }

      /* Centering! */
      .container {
        text-align: center;
      }

      .doggies {
        /* Rounding things out */
        border-radius: 50%;
        max-width: 300px;
      }
    </style>
  </head>

  <body>
    <div class="container">
      <div>
        <img class="doggies" src="../res/doggies.jpg">
      </div>

      <div>
        <h1>Here's a title!</h1>
        <p>Here's some text content!</p>
        <h2>Here's a smaller title!</h2>
        <p>Here's some more text content!</p>
      </div>
    </div>
  </body>
</html>

Step 5

Putting in a background

<!DOCTYPE html>

<html>
  <head>
    <style>
      body {
        font-family: Helvetica, Arial, sans-serif;

        /* Background stuffs */
        min-height: 100vh;
        background-image: url("../res/CSE.png");
        background-position: center top;
        background-repeat: no-repeat;
        background-size: cover;
      }

      .container {
        text-align: center;
      }

      .doggies {
        border-radius: 50%;
        max-width: 300px;
      }

      /* Let's add some spacing */
      #header-picture {
        padding: 30px 0;
      }

      /* Add some color to the text, too */
      #coolbox {
        display: inline-block;
        background-color: rgba(246,255,255,.9);
        padding: 30px;
      }
    </style>
  </head>

  <body>
    <div class="container">
      <div id="header-picture">
        <img class="doggies" src="../res/doggies.jpg">
      </div>

      <div id="coolbox">
        <h1>Here's a title!</h1>
        <p>Here's some text content!</p>
        <h2>Here's a smaller title!</h2>
        <p>Here's some more text content!</p>
      </div>
    </div>
  </body>
</html>

Step 6

Adding a splash of design

<!DOCTYPE html>

<html>
  <head>
    <style>
      body {
        font-family: Helvetica, Arial, sans-serif;
        min-height: 100vh;

        background-image: url("../res/CSE.png");
        background-position: center top;
        background-repeat: no-repeat;
        background-size: cover;
      }

      .container {
        text-align: center;
      }

      .doggies {
        border-radius: 50%;
        max-width: 300px;
      }

      #header-picture {
        padding: 30px 0;
      }

      #coolbox {
        display: inline-block;
        background-color: rgba(246,255,255,.9);
        padding: 30px;
      }

      /* Shadows! */
      .shadow {
        -moz-box-shadow: 0 6px 12px rgba(0,0,0,.4);
        -webkit-box-shadow: 0 6px 12px rgba(0,0,0,.4);
        box-shadow: 0 6px 12px rgba(0,0,0,.4);
      }

      /* A class to apply round corners */
      .round-corners {
        border-radius: 10px;
      }

      /* A class for the subtitle */
      .subtitle {
        color: #777;
      }

      /* Highlight this text in particular */
      .woah {
        color: #F00;
      }
    </style>
  </head>

  <body>
    <div class="container">
      <div id="header-picture">
        <img class="doggies shadow" src="../res/doggies.jpg">
      </div>

      <div id="coolbox" class="round-corners shadow">
        <h1>Here's a title!</h1>
        <p class="subtitle">
          Here's some <span class="woah">text</span> content!
        </p>
        <h2>Here's a smaller title!</h2>
        <p>Here's some more text content!</p>
      </div>
    </div>
  </body>
</html>
